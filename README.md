The Heart Failure Prediction System is a machine learning web application designed to predict the risk of death due to heart failure using clinical patient data.

The system uses a trained machine learning model to analyze patient health parameters and provide a prediction through an interactive web interface.

This project demonstrates a complete end-to-end machine learning pipeline, including:

Data ingestion

Data validation

Data transformation

Model training

Model evaluation

Model deployment using FastAPI

The main objectives of this project are:

Provide a user-friendly web interface

Demonstrate production-style ML project structure

The model uses the following clinical features to make predictions.

Feature	Description
